# X Undo

Undo/Redo for X (Twitter) timeline. Never lose that tweet you accidentally scrolled past!

## Features

- **Back/Forward** - Navigate through your timeline history like browser back button
- **History List** - Jump to any previous state directly
- **Search** - Find tweets in your saved history by keyword
- **Keyboard Shortcuts** - Ctrl+Z / Ctrl+Y (customizable)
- **Scroll Position** - Returns you to the exact spot
- **Theme Support** - Works with Dark, Dim, and Light modes

## Performance

- Saves only on changes (MutationObserver)
- Compressed storage (LZ-string)
- IndexedDB for large history
- Lightweight and fast

## Privacy

- All data stored locally
- No external servers
- No data collection
- No tracking
